# Members

Get all member of one of my teams

**URL** : `/team/members/<team_name>`

**Method** : `GET`

**Auth required** : YES

## Success Response

**Code** : `OK`

**Content example**

```json
[
    {
        "username": "tester_static",
        "time": "-120",
        "picture": "new profile picture",
        "country_code": "RU"
    },
    {
            "username": "killian",
            "time": "-180",
            "picture": "pp",
            "country_code": "US"
        }
]
```
